📖 How to use this Explorer

🚀 Getting Started

Click Fetch Data from Backend to automatically download the concept_map_for_lesson_generation.json file from the server.

📋 Table View

  • Filter by chapter, CEFR level, kind, and more using the left sidebar
  • Click any row to open the detail panel on the right
  • Click column headers to sort
  • Use the search box for instant text filtering

🕸 Network View

  • Click a node to see its full details in the right panel
  • Drag nodes to rearrange the graph
  • Scroll/pinch to zoom; drag background to pan
  • Path Finder: Set a "From" and "To" node to find the shortest connection path between them
  • Hop Navigator: Click a node, then use "Set as From / To" in the panel to find paths
  • Use the left sidebar filters to reduce nodes shown

🌐 Radial View

  • See concepts arranged in a sunburst / radial layout by chapter or level
  • Click segments to drill in

📊 Stats View

  • Corpus-wide statistics: distribution by chapter, level coverage, asterisk rates
